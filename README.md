# Room Characteristics Storage System CPP
 Simple console based sotrage system, used for documenting rooms within an apartment. 

## Description
 This was the one of the final projects for CS305 Programming Languages. Its a console based application made to store data about rooms within an apartment. The user can add as many rooms as the wish, as well as characteristics about that room. Afterwards, it is possible to delete or make edits to said room.

## How to Use
 Just clone the repository to your local machine, and open the .cbp file. Afterwards just run.
